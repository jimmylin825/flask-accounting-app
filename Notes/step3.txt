✅ 步驟三：查詢與篩選功能
🎯 本階段目標：

讓使用者可以在頁面中 依照「起始日期」、「結束日期」與「類型」 來查詢收支資料。
🔧 你要嘗試實作的內容：

    在 expenses.html 加上一個查詢表單

        表單欄位：開始日期、結束日期、類型（income / expense / all）

        表單 method 使用 GET

    在 expenses.py 中的 route 讀取 query string

        使用 request.args.get() 取得表單輸入

        依據條件過濾 Expense 查詢結果

    回傳篩選後的結果到模板顯示

💡 提示：

    使用者送出的表單會帶著參數例如：

/expense?start=2024-01-01&end=2024-01-31&type=income

你可以用類似這樣的條件組合篩選：

if start_date:
    query = query.filter(Expense.date >= start_dt)

類型選擇要用下拉選單 <select>，方便選擇 income 或 expense。
